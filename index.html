Privacy Policy for 9K Extension Bodyguard

9K Extension Bodyguard helps users review installed Chrome extensions locally. It explains declared permissions, site access patterns, and local extension changes in plain language.

Overview

9K Extension Bodyguard is designed to keep data on the user's device. It reviews installed extensions only, and its reports are based on declared permissions and extension metadata exposed by Chrome, not live extension behavior.

The extension does not sync data to a server and does not send extension review data outside the device.

Data Processed Locally

9K Extension Bodyguard processes extension metadata provided by Chrome's extension management API and local settings saved by the extension.

Installed extension information

  • Installed extension names.
  • Extension versions.
  • Enabled or disabled state.
  • Install type when Chrome provides it.
  • Extension icons when Chrome provides them.
  • Declared API permissions.
  • Declared site access patterns.
  • Locally observed extension installation, update, enable/disable, permission, and site-access changes.

Local user data

  • Local notes.
  • Trust marks.
  • Welcome or intro screen state.
  • Language preference.
  • Local permission review snapshots.
  • Recent extension activity and change history.

Where Data Is Stored

Data is stored in chrome.storage.local on the user's device. 9K Extension Bodyguard does not sync this data to a server, sell it, or share it with third parties.

What 9K Extension Bodyguard Does Not Do

9K Extension Bodyguard does not:

  • Scan websites.
  • Read page content.
  • Read browsing history.
  • Read cookies.
  • Use content scripts.
  • Request host permissions.
  • Monitor live browsing behavior.
  • Use analytics.
  • Show ads.
  • Call remote APIs.
  • Make AI calls.
  • Sell user data.
  • Share user data with third parties.
  • Sync data to a server.

Requested Chrome Permissions

9K Extension Bodyguard requests only the Chrome permissions required for its local extension review features.

management

Used to read installed extension names, versions, enabled state, icons, install type, declared permissions, and site access so the dashboard can explain extension access locally.

storage

Used to save local notes, trust marks, welcome screen state, language preference, local permission review snapshots, and recent extension activity and change history on this device.

alarms

Used to schedule local extension checks and toolbar badge refreshes. Alarm events only trigger processing on the user's device and do not transmit data.

Data Sharing and Sale

9K Extension Bodyguard does not sell or transfer user data to third parties.

It does not use data for advertising, profiling, creditworthiness, unrelated purposes, analytics, remote reporting, or AI processing.

Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Information is used only to provide 9K Extension Bodyguard's local review features. It is not transferred for advertising, profiling, creditworthiness, or unrelated purposes, and it is not made available for human review.

User Control

Users can:

  • Edit or remove local notes.
  • Add or remove trust marks.
  • Reset the welcome or intro screen state.
  • Reset local permission-review history.
  • Mark permission changes as reviewed.
  • Clear locally stored data from Settings.

Uninstalling 9K Extension Bodyguard removes its extension data according to Chrome's extension storage behavior.

Limitations

9K Extension Bodyguard reviews declared permissions and site access exposed by Chrome.

It does not inspect live extension behavior, verify developer reputation, classify extensions as objectively safe or unsafe, detect malware, block attacks, or provide real-time protection.

Changes to This Policy

Future changes to this policy will be documented with the applicable extension version and update date.
